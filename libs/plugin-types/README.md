# Penpot plugin-types

This repository contains the typings for the Penpot Plugin API.

### Getting started

Install the package:

```bash
npm install @penpot/plugin-types
```

Configure `tsconfig.json`

```json
"typeRoots": [
  "./node_modules/@types",
  "./node_modules/@penpot"
],
"types": ["plugin-types"],
```
